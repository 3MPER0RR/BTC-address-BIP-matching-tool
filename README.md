# BTC address BIP maching tool

## build 
cd btc address BIP maching tool
cargo build 
cargo run

## usage

cd btc address BIP maching tool
cargo run

When prompted:
Insert your 12 or 24-word mnemonic (BIP39)
Leave blank or insert the passphrase (BIP39 seed extension), if used

## OUTPUT The tool derives and prints Bitcoin addresses for:
BIP44 – Legacy (P2PKH)

BIP49 – SegWit (P2SH-P2WPKH)

BIP84 – Native SegWit (Bech32)
